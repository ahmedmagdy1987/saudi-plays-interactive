# public/brand/

Drop **verified official** logo files here (SVG preferred, else high-resolution
transparent PNG), then reference them with root-relative paths (e.g.
`/brand/gea.svg`) and update `docs/brand-assets.md`.

No files are bundled yet: none of the referenced entities' official marks could be
verified to the required provenance standard in this build, so the UI uses
localized text labels instead (see `docs/brand-assets.md`). Never redraw,
AI-generate, trace, recolor, or reproportion an official logo.
